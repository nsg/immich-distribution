# About configuration

By default you should have a working installation of Immich, but you may like to configure it a bit.

Immich Distribution focuses on ease of install, not to be fully configurable so everything from upstream Immich is not configurable. But feel free to open an issue if you like to have an option added.

![](/assets/admin-display.png)

Most Immich Settings can be changed under `Administration` in Immich from a user with the appropriate permissions. The documentation provided here covers settings that can't be changed from Immich Administration, or changes specific for Immich Distribution. For step-by-step instructions for common tasks, see the [guides](/guides/) section.

:material-chat-alert-outline: The configuration options are organized in separate sub pages under this section.
